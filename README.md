## Cara Menjalankan
- Clone Repository ini
- Masuk Kedalam directory utama dan jalankan
```
npm install
```
atau
```
npm install -f
```
- tunggu hingga proses selesai
- jalankan program dengan cara
```
npm run start
``` 
- pilih android atau web sesuai yang anda inginkan
